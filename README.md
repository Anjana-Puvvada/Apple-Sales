# iPhone Sales Analysis in India

## Overview
This project analyzes iPhone sales in the Indian market, focusing on factors such as customer ratings, pricing, and discounts. The analysis provides insights into consumer preferences and trends, aiming to guide strategic decisions for businesses and retailers operating in the Indian iPhone market.

## Key Features
- **Identifies the top-rated iPhones in India based on customer ratings.**
- **Examines the relationship between sale price and customer ratings.**
- **Investigates the impact of discount percentage on customer ratings.**

## Methodology
- Utilizes Python for data manipulation and analysis.
- Uses the provided dataset ("apple_products.csv") from Kaggle for analysis.
- Employs Pandas, NumPy, and Plotly Express for data processing and visualization.

## Key Findings
1. **"APPLE iPhone 8 Plus (Gold, 64 GB)" emerges as the top-rated iPhone with the highest number of ratings and reviews.**
2. **Lower-priced iPhones tend to receive higher ratings, indicating consumer preference for affordability.**
3. **iPhones with higher discounts attract more ratings, suggesting the importance of promotional strategies.**

## Conclusion
Understanding consumer preferences and market dynamics is crucial for success in the Indian smartphone market. This analysis provides actionable insights that businesses can use to make informed decisions about product offerings and pricing strategies.
